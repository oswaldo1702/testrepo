En este video, vamos a revisar las características de alto nivel del lenguaje de programación Python. Python es un lenguaje potente. Es, con mucho, el lenguaje de programación más popular para la ciencia de datos. Según la encuesta de Kaggle Data Science y Machine Learning de 2019, el 75% de los más de 10.000 encuestados de todo el mundo informaron de que utilizaban Python de forma regular. Glassdoor informó que en 2019 más del 75% de los puestos de ciencia de datos listados incluían Python en sus descripciones de trabajo. Cuando se le preguntó qué idioma debería aprender primero un aspirante a científico de datos, la mayoría de los científicos de datos dicen Python. Probablemente estés pensando, ¿por qué demonios Python es tan popular? Bueno, comencemos con las personas que usan Python. Si ya sabe cómo programar, entonces Python es genial para usted porque usa una sintaxis clara y legible. Puede hacer muchas de las cosas que está acostumbrado a hacer en otros lenguajes de programación, pero con Python puede hacerlo con menos código. Si quieres aprender a programar, también es un gran idioma de inicio debido a la enorme comunidad global y la riqueza de documentación. De hecho, varias encuestas diferentes en 2019 encontraron que más del 80% de los profesionales de datos en todo el mundo utilizan Python. Python es útil para muchas situaciones, incluyendo ciencia de datos, IA y aprendizaje automático , desarrollo web y dispositivos de IoT como Raspberry Pi. Las grandes organizaciones que usan Python en gran medida incluyen IBM, Wikipedia, Google, Yahoo! , CERN, NASA, Facebook, Amazon, Instagram, Spotify y Reddit. Python es un poderoso lenguaje de programación de propósito general que puede hacer muchas cosas. Es ampliamente apoyado por una comunidad global y dirigido por la Python Software Foundation. 1. Python es un lenguaje de programación de propósito general de alto nivel que se puede aplicar a muchas clases diferentes de problemas. 2. Cuenta con una gran biblioteca estándar que proporciona herramientas adecuadas para muchas tareas diferentes, incluyendo, entre otras, bases de datos, automatización, raspado web, procesamiento de texto, procesamiento de imágenes , aprendizaje automático y análisis de datos. 3. Para la ciencia de datos, puede usar las bibliotecas de computación científica de Python como Pandas, NumPy , SciPy y Matplotlib. 4. Para la inteligencia artificial, tiene TensorFlow, PyTorch, Keras y Scikit-Learn. 5. Python también se puede usar para el procesamiento de lenguaje natural (PNL) usando el kit de herramientas de lenguaje natural (NLTK). Otro gran punto de venta es la comunidad de Python, que tiene una historia bien documentada de allanar el camino para los esfuerzos de diversidad e inclusión en la industria tecnológica en su conjunto. El lenguaje Python tiene un código de conducta ejecutado por la Python Software Foundation que busca garantizar la seguridad y la inclusión para todos, tanto en las comunidades de Python en línea como en persona. También hay comunidades como PyLadies que buscan crear espacios para que las personas interesadas en Python aprendan en entornos seguros e inclusivos. PyLadies es un grupo internacional de mentores con un enfoque en ayudar a más mujeres a convertirse en participantes y líderes activos en la comunidad de código abierto de Python.
